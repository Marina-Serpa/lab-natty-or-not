# Podcast - Líderes em Ação - Natty or Fake Natty?

## 📒 Descrição
Este podcast foi desenvolvido por uma inteligência artificial avançada, tem como foco a liderança corporativa. Abordamos temas essenciais como as características fundamentais de um líder, a diferença entre líder e chefe, exemplos de líderes inspiradores e como desenvolver habilidades de liderança.

## 🤖 Tecnologias Utilizadas
[https://openai.com/pt-BR/](https://copilot.cloud.microsoft/?fromcode=cmc&redirectid=1C461B030BCC465EAA91B495E9213CD1&auth=2)

[elevenlabs](https://elevenlabs.io/)

## 🧐 Processo de Criação
Para elaborar este podcast sobre liderança, contei com a ajuda do Copilot (Microsoft). Ele foi fundamental na criação do roteiro e texto, a organização dos temas foi realizada totalmente pela IA. Para transformar o texto em audio (text to speech), utilizei uma ferramenta de inteligência artificial, elevenlabs. Essas tecnologias combinadas garantiram a qualidade e o impacto deste material.

## 🚀 Resultados
Criação de um podcast com mais de 3 minutos.
